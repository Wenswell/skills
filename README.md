# Local Codex Skills

This repository stores local, portable Codex skills.

## Skills

- `.agents/skills/codex-session-profiler/`: Analyze Codex session history into a coding-agent profile, reusable prompt, memory notes, and breadcrumb-style expression details.

## Notes

- Generated outputs such as `data/` and `reports/` are ignored.
- The runnable scripts live inside each skill's `scripts/` directory.
- Detailed workflow notes live inside each skill's `references/` directory.
